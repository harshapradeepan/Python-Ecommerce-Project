🛒 Python Ecommerce Data Analysis

📌 About
Analysis of ecommerce data using Python and MySQL. 
Explores customer behavior, sales trends, and order patterns.

🔍 Key Analyses
1. List all unique cities where customers are located
2. Count total orders placed in 2017 (90,202 orders!)
3. Total sales per product category (74 categories)
4. Percentage of orders paid in installments (99.99%)
5. Customer count by state (with bar chart visualization)

🛠️ Tools & Technologies
- Python
- MySQL (mysql-connector)
- Pandas
- Matplotlib
- Seaborn
- NumPy
- Jupyter Notebook

▶️ How to Run
1. Clone the repo
2. Set up MySQL database named `ecommerce`
3. Install dependencies: `pip install pandas matplotlib seaborn mysql-connector-python numpy`
4. Open `ecommerce.ipynb` in Jupyter Notebook
5. Run all cells
